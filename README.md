# TKIDLE

可视化 Tkinter GUI 界面生成工具 — 通过鼠标拖拽摆放控件，一键输出可运行的 Python 代码。

A visual GUI builder for Python Tkinter — drag and drop widgets, then export a runnable Python file.

## 运行 / Usage

```bash
python start.py
```

Project → New 新建项目，Component 菜单添加控件，Output → Output GUIofProject.py 导出代码。

## 功能 / Features

- 支持 6 种控件：Button, Label, Text, Entry, Checkbutton, Radiobutton
- 鼠标拖拽定位 & 缩放控件大小
- 右键菜单编辑属性（名称、文本、字体、颜色、关联变量）
- 生成的 `.py` 文件可直接运行

## 环境 / Requirements

- Python 3.x（自带 tkinter）
